Help

使用帮助

适合 PPTX、DOCX 和可复制文字的 PDF 翻译、校对、导出。

基本流程

  1. 选择或拖入 PPTX、DOCX、PDF 文件；旧版 PPT/DOC 请先另存为新版格式。
  2. 在“参数”里选择翻译方向、模型和 DeepSeek API Key。
  3. 点击“自动翻译”，完成后可逐段修改译文。
  4. 翻译中同一个按钮会变成“停止翻译”，可随时中断并保留已完成译文。
  5. 用“预览译文”检查版式，再导出或分享翻译文件。

主要功能

  • 支持中文、英文、日文、韩文、法文、德文、西班牙文等多语种目标语言。
  • 支持一次选择多个文件，使用“批量翻译并保存”顺序处理。
  • PPTX 预览里可拖动文本框、调整大小、切换单行/换行和单段字号。
  • PDF 会尽量保留页面背景、图片、表格线，并在原文字位置覆盖写入译文。
  • 生成后的文件会进入“文件库”，可再次下载、分享或删除。

手机使用

  • 右上角菜单可切换“翻译、参数、输出、文件库”分区。
  • 刷新或误触下拉刷新后，会自动恢复上次未导出的当前草稿。
  • 长文件翻译或 PDF 导出时请保持页面打开，避免锁屏或切到后台。
  • 右上角钥匙图标可进入超级管理员登录和授权手机号管理。
